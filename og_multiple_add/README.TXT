INTRODUCTION
------------

"OG Add User to Multiple" is a Drupal module that provides a form to add one user to multiple OG groups in a batch.
On the form you can fill in the user name of the user you want, and choose from the groups you want to add the user to, from the list, or choose any content type (which is defined as a group) and the user will be added into all the groups of that content type.
You can also choose the role you want to give to the chosen user. If the role is does not exist in a specific group, the user will only be added as a member.
