INTRODUCTION
------------

"OG Add User to Multiple" is a Drupal module that provides a form to add one user to multiple OG groups in batch.
In that form you fill the user you want, and choose from the list the groups you want to add into, or you can 
choose any content type (which defined as group) and the user will be added into all the groups of that content type.
Additionally you can choose the role you want to give to the chosen user. If the role is not exist in a specific group,
the user will be added just as a member.
